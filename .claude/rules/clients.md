# クライアント情報

## 天弥堂
- 業種: 香り・占い・陶芸・Webデザインの複合サロン
- 設定: clients/天弥堂/content_profile.json（トーン・ハッシュタグ・SNSフォーマット）
- SNS設定: clients/天弥堂/sns_config.json（各SNSの有効/無効・cronタイミング・アセット）
- テンプレート: clients/天弥堂/templates/{x,instagram,threads,facebook}.txt
- アセット画像: clients/天弥堂/assets/images/{kaori,uranai,tougei,design}/
- BGM: clients/天弥堂/assets/bgm/meditation.mp3

## SNSごとの方針
| SNS | ハッシュタグ | 文字数 | 特記 |
|-----|------------|--------|------|
| X | 2個 | 120字以内 | 問いかけ締め、動画付き |
| Instagram | 3〜4個 | 制限なし | Hook→本文→CTA構成 |
| Threads | 1個 | 200字以内 | 会話誘発、返信促進 |
| Facebook | 1個 | 制限なし | 体験談＋問いかけ |
| TikTok | 4個 | Hook重視 | 最初3秒で止める一言 |
| YouTube Shorts | 4個 | Hook重視 | 検索キーワード重視 |
